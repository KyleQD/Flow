## System Overview

### What's New in Phase 2

The Phase 2 onboarding system introduces:

- **Automated Workflow Engine**: 9-stage workflow from invitation to team assignment
- **Multi-Channel Notifications**: Email, SMS, push, and in-app notifications
- **Visual Workflow Tracking**: Kanban board and progress visualization
- **Advanced Analytics**: Performance metrics and bottleneck identification
- **Smart Automation**: Automatic stage progression and reminders

### Key Features

1. **Workflow Dashboard**: Overview of all onboarding processes
2. **Kanban Board**: Visual management of candidates by status
3. **Workflow Visualizer**: Detailed tracking of individual workflows
4. **Notification Center**: Centralized notification management
5. **Analytics Dashboard**: Performance insights and reporting

## Admin Training

### Accessing the Onboarding System

1. **Navigate to Admin Dashboard**
   - Go to `/admin/dashboard`
   - Click on "Staff Management" tab
   - Select "Onboarding" from the sidebar

2. **System Overview**
   - **Dashboard Tab**: High-level statistics and quick actions
   - **Candidates Tab**: Manage individual candidates
   - **Templates Tab**: Configure onboarding templates
   - **Workflows Tab**: Visual workflow management

### Dashboard Overview

#### Key Metrics
- **Total Candidates**: Number of active candidates
- **Pending Review**: Candidates awaiting admin review
- **In Progress**: Candidates currently in onboarding
- **Completed**: Successfully onboarded candidates
- **Average Progress**: Overall completion rate

#### Quick Actions
- **Add Existing User**: Invite current users to onboarding
- **Invite New User**: Send invitations to new users
- **View Analytics**: Access detailed performance reports
- **Manage Templates**: Configure onboarding forms

### Managing Candidates

#### Adding Candidates

1. **Add Existing User**
   - Click "Add Existing User" button
   - Search for user by email or name
   - Select appropriate template
   - Set initial stage and priority
   - Click "Add to Onboarding"

2. **Invite New User**
   - Click "Invite New User" button
   - Enter email and/or phone number
   - Select template and role
   - Customize invitation message
   - Send invitation

#### Candidate Management

1. **List View**
   - View all candidates in a table format
   - Sort by status, date, or name
   - Filter by department or stage
   - Bulk actions for multiple candidates

2. **Kanban View**
   - Visual board with columns for each status
   - Drag and drop candidates between stages
   - Quick status updates
   - Color-coded priority levels

### Workflow Management

#### Understanding Workflow Stages

The system includes 9 automated stages:

1. **Invited**: Candidate has been invited
2. **Application Started**: Candidate began onboarding
3. **Documents Submitted**: Required documents uploaded
4. **Background Check**: Background verification in progress
5. **Interview Scheduled**: Interview has been arranged
6. **Interview Completed**: Interview finished
7. **Reference Check**: References being contacted
8. **Final Review**: Admin review pending
9. **Approved**: Candidate approved and added to team

#### Managing Workflows

1. **Workflow Visualizer**
   - Navigate to "Workflows" tab
   - View all active workflows
   - See progress for each candidate
   - Identify bottlenecks and delays

2. **Stage Management**
   - Click on any workflow to view details
   - See current stage and next steps
   - Manually advance stages if needed
   - Add notes and comments

3. **Automated Actions**
   - System automatically sends notifications
   - Reminders for incomplete tasks
   - Escalation for stuck processes
   - Status updates to relevant parties

### Template Management

#### Creating Templates

1. **Template Builder**
   - Navigate to "Templates" tab
   - Click "Create New Template"
   - Add form fields and requirements
   - Set validation rules
   - Configure notifications

2. **Field Types**
   - **Personal Information**: Name, contact details
   - **Address Fields**: Home and emergency addresses
   - **Emergency Contact**: Contact information
   - **Bank Information**: Payment details
   - **Tax Information**: Tax forms and IDs
   - **ID Documents**: Government-issued IDs
   - **Custom Fields**: Venue-specific requirements

#### Template Configuration

1. **Required Fields**
   - Mark fields as required or optional
   - Set validation rules (email format, phone format)
   - Add help text and instructions
   - Configure conditional logic

2. **Notifications**
   - Set up automatic notifications
   - Configure reminder schedules
   - Define escalation rules
   - Customize message templates

## Troubleshooting

### Common Issues

#### Can't Access System
1. **Check Email**: Verify invitation email
2. **Reset Password**: Use "Forgot Password" link
3. **Clear Browser Cache**: Clear cookies and cache
4. **Try Different Browser**: Use Chrome, Firefox, or Safari
5. **Contact Admin**: Reach out for help

#### Can't Upload Documents
1. **Check File Size**: Ensure files are under 10MB
2. **Check File Format**: Use PDF, JPG, or PNG
3. **Check Internet**: Ensure stable connection
4. **Try Different File**: Use alternative document
5. **Contact Support**: Get technical help

#### Not Receiving Notifications
1. **Check Spam Folder**: Look in junk/spam folder
2. **Verify Contact Info**: Ensure email/phone is correct
3. **Check Settings**: Verify notification preferences
4. **Add to Contacts**: Add venue to contacts
5. **Contact Admin**: Update contact information

#### Workflow Stuck
1. **Check Requirements**: Ensure all fields completed
2. **Upload Documents**: Provide required documents
3. **Contact Admin**: Ask about next steps
4. **Check Timeline**: Review estimated completion
5. **Request Update**: Ask for status update
