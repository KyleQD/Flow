# Staff Management System - User Guide

## üìã **Overview**

The Staff Management System is a comprehensive platform for managing event staffing, from job postings to team management. This guide covers all features and workflows for administrators.

## üöÄ **Quick Start**

### **Accessing the System**
1. Navigate to `/admin/dashboard/staff`
2. Ensure you have admin privileges
3. Select your venue from the dropdown

### **Main Dashboard**
The dashboard provides an overview of:
- **Onboarding Statistics**: Candidates in progress, completed, pending
- **Job Postings**: Published, draft, and closed positions
- **Staff Management**: Active staff, departments, performance metrics

## üìù **Job Posting Management**

### **Creating a Job Posting**
1. Click **"Create Job Posting"** button
2. Fill in the required fields:
   - **Title**: Position title (e.g., "Security Guard")
   - **Department**: Department (e.g., "Security")
   - **Position**: Specific role (e.g., "Event Security")
   - **Description**: Detailed job description
   - **Requirements**: Skills and qualifications needed
   - **Compensation**: Pay rate and benefits
   - **Schedule**: Work hours and availability
3. Click **"Create Posting"**

### **Managing Job Postings**
- **View All Postings**: See all job postings in the "Job Postings" tab
- **Edit Posting**: Click edit icon to modify details
- **Pause/Resume**: Toggle posting status
- **Close Posting**: Mark as filled or closed
- **View Applications**: See all applicants for each posting

## üìã **Application Review**

### **Reviewing Applications**
1. Navigate to **"Applications"** tab
2. View all applications with status indicators:
   - **Pending**: New applications to review
   - **Reviewed**: Applications under consideration
   - **Approved**: Accepted candidates
   - **Rejected**: Declined applications

### **Application Actions**
- **View Details**: Click on application to see full details
- **Update Status**: Change application status
- **Send Message**: Communicate with applicants
- **Bulk Actions**: Select multiple applications for batch processing
- **Export Data**: Download application data for reporting

### **Application Information**
Each application shows:
- **Applicant Details**: Name, email, phone
- **Experience**: Years of experience, skills
- **Form Responses**: Answers to job-specific questions
- **Rating**: Automated or manual rating
- **Application Date**: When application was submitted

## üéØ **Onboarding Management**

### **Onboarding Workflow**
1. **Invitation**: Send onboarding invitation to approved candidates
2. **Document Collection**: Collect required documents (ID, background check, etc.)
3. **Training**: Complete required training modules
4. **Verification**: Verify all documents and training completion
5. **Completion**: Mark onboarding as complete

### **Onboarding Dashboard**
- **Candidate Cards**: Visual overview of all onboarding candidates
- **Progress Tracking**: Real-time progress indicators
- **Stage Management**: Move candidates through onboarding stages
- **Document Management**: Upload and verify required documents

### **Onboarding Actions**
- **Update Progress**: Track completion of onboarding steps
- **Send Messages**: Communicate with candidates during onboarding
- **Upload Documents**: Add required documents to candidate profiles
- **Complete Steps**: Mark individual onboarding steps as complete
